Чтобы написать тесты:
1. Создаем .cpp файл с названием файла "Test" + который хотим тестировать (Например: Config.cpp -> TestConfig.cpp)
2. Добавляем хедер в созданный файл ./unit_test/include/acutest.hpp
3. Пишем тесты-функции при помощи документации (https://github.com/mity/acutest)
4. После написания всех функиций в конце файла создаем массив следующего типа:
```
TEST_LIST = {
    { "название теста", function_name_1 },
    { "название другого теста",     function_name_2 },
    { "название еще какого теста",    function_name_3 },
    { "название ну ты понял теста",    function_name_n },
    { nullptr, nullptr }
};
```
5.  Запускаем make или в папке ./unit_test - чтобы проверить только тесты
